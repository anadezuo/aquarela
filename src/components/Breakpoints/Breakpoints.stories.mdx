# Breakpoints

Breakpoints for responsive layout:

<table>
  <tr>
		<th>Extra Small (sx)</th>
		<td>Default </td>
  </tr>
  <tr>
		<th>Small (sm)</th>
		<td> >= {BreakpointsSize.sm} px</td>
  </tr>
  <tr>
		<th>Medium (md)</th>
		<td> >= {BreakpointsSize.md} px</td>
  </tr>
	<tr>
		<th>Large (lg)</th>
		<td> >= {BreakpointsSize.lg} px</td>
  </tr>
	<tr>
		<th>Extra Large (xl)</th>
		<td>>= {BreakpointsSize.xl} px</td>
  </tr>
</table>


Using in styled component:

```js
import { breakAt, BreakpointsSize} from "components/Breakpoints/Breakpoints"

const Title = style.h1`
  font-size: 1rem;

	${breakAt(BreakpointsSize.md)}{
		font-size: 2rem;
	}
`

```
